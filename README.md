# Mini File System Simulator (Operating Systems Project)

## Overview
This project is based on the research paper:
**"A Lightweight File System Simulator for Teaching Operating Systems" (2023)**

It demonstrates how file systems manage:
- Block allocation  
- File creation  
- Directory structure  
- Data writing & reading  
- Storage management  

## Requirements
- Python 3.8 or above
- Windows, Linux, or Mac

## How to Run
Open terminal/CMD in project folder and type:
1. Open terminal/CMD in project folder
2. Run the command:
   python mini_fs.py
3. Follow the on-screen instructions to create, write, read, or delete files
